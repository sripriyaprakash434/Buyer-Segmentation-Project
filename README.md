# Buyer Segmentation Project

## Overview
This project performs customer segmentation using K-Means Clustering on real estate buyer data.

## Dataset
- clients.csv
- properties.csv

## Features Used
- Age
- Sale Price
- Floor Area (sqft)
- Satisfaction Score

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Project Workflow
1. Data Cleaning
2. Data Merging
3. Feature Engineering
4. Data Scaling
5. K-Means Clustering
6. Cluster Analysis
7. Data Visualization

## Cluster Summary

| Cluster | Description |
|----------|-------------|
| 0 | Low satisfaction buyers |
| 1 | High satisfaction senior buyers |
| 2 | Premium property investors |
| 3 | Young mid-range buyers |
